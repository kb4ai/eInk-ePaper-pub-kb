**FAQs about Paperlike [Mac Edition]**

**1. What sizes are available for the [Mac Edition]?**

A: The [Mac Edition] is available for 10.3 - inch, 13.3 - inch, and 25.3 - inch ultra-high refresh rate E - ink display series.

**2. Which Apple devices are compatible with the [Mac Edition]?**

A: Currently, it supports Apple devices with M1, M2, M3, and M4 chips. However, it still doesn’t support Apple computers with Intel chips, iPhones, or iPads.

**3. Is there a difference in refresh rate between the [Mac Edition] and non - Mac Edition?**

A: The refresh rate of the [Mac Edition] is as fast as the corresponding non - Mac Edition:

10.3 - inch ultra - high refresh rate E - ink display [Mac Edition]: 60Hz LCD - level ultra - high refresh rate

13.3 - inch ultra - high refresh rate E - ink display [Mac Edition]: 37Hz ultra - high refresh rate

25.3 - inch ultra - high refresh rate E - ink display [Mac Edition]: 33Hz ultra - high refresh rate

**4. What precautions should be taken when using the [Mac Edition]?**

A: To connect the [Mac Edition] display to a Mac computer, you must first install and run the latest Paperlike client (Mac V2.0) on the computer. The client can be downloaded at www.dasung.com.

**5. Can the [Mac Edition] connect to Windows systems?**

A: Yes, but to connect the [Mac Edition] display to a Windows computer, you must install and run the latest Paperlike client (Windows V2.0) on the computer in advance. The latest Windows V2.0 client is expected to be released around the end of June, and you can download it at [大上科技-DASUNG](https://www.dasung.com/h-col-112.html)

**6. If I buy a non - Mac version of the "Revolutionary" series display, can I connect it to Apple devices by installing and running the Paperlike client?**

A: No. Non - Mac version displays are still strictly prohibited from connecting to Apple devices.
